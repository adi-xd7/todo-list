# 📝 Todo List App

A simple and clean **Todo List web application** built using **HTML, CSS, and JavaScript** to manage daily tasks efficiently.

---

## 🚀 Features

* ➕ Add new tasks with a due date
* 📅 Select date using a date picker
* ❌ Delete tasks easily
* ⚠️ Input validation (prevents empty tasks)
* 🎨 Clean and modern UI design
